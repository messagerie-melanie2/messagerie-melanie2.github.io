    DS du Bnum

    Design System du Bnum – Composants UI réutilisables pour les applications du Bnum.

    Présentation

    Ce projet est un Design System destiné à centraliser et standardiser les composants d’interface utilisateur pour les applications du Bnum. Il fournit des composants web, des styles, des icônes et des thèmes pour garantir la cohérence graphique et l’accessibilité.

    Installation

    Installez ds-bnum.min.js dans votre projet

    Utilisation

    Incluez-les dans votre HTML :

    <script type="module" src="ds-module-bnum.min.js"></script>

    Structure du projet

    • components/ : Composants UI (boutons, cartes, icônes…)
    • core/ : Styles de base, tokens, utilitaires
    • themes/ : Thèmes CSS (dark, light)
    • build/ : Fichiers générés lors du build
    • docs/ : Documentation et exemples
    • src/ : Point d’entrée principal
    • package.json : Dépendances et scripts

    Scripts

    • npm run build : Génère la librairie
