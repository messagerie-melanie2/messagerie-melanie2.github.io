      Décorateur d'accesseur automatique pour gérer des instances de JsEvent.

      Ce décorateur transforme un accesseur (auto-accessor) en une propriété gérée, assurant une instanciation unique (Singleton par propriété) et une gestion du cache. Il empêche également l'écrasement accidentel de l'événement via le setter.

      Type Parameters

      • TCallback extends Function

        Signature de la fonction callback de l'événement.

      • This = any

        Type de l'instance de la classe parente.

      Parameters

      • Optionalinitializator: Nullable<CallbackInitializator<TCallback, This>>

        (Optionnel) Fonction exécutée une seule fois à la création de l'événement pour le configurer.

      • Optionaloptions: ListenerDefaultOptions

        (Optionnel) Configuration du comportement du listener (lazy loading, type d'événement).

      Returns (
          _target: ClassAccessorDecoratorTarget<This, JsEvent<TCallback>>,
          context: ClassAccessorDecoratorContext<This, JsEvent<TCallback>>,
      ) => ClassAccessorDecoratorResult<This, JsEvent<TCallback>>

      Le décorateur d'accesseur de classe conforme à la norme ES Decorators (Stage 3).

      Throws

      Si une tentative d'assignation (setter) est effectuée sur la propriété décorée.

      Example

      class MyComponent {
      @Listener((evt, instance) => evt.attach(instance.handleAction), { lazy: true })
      accessor onAction: JsEvent<(val: string) => void>;

       private handleAction(val: string) {
         console.log(val);
       }
      }
